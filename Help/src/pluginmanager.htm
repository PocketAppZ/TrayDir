Plugin Manager

The plugin manager can be used to add new plugins to list of known plugins. Plugins can be configured to point to a specific file in the current user's file system, then parameters can be defined so they can vary by menu item.

Example

For example, I want to add Google Chrome as a plugin, with one parameter being the website's URL I want to open. To do this I would do the following:

Create the plugin

  1. Open Plugin Manager.
  2. Click Add button.
  3. Click Browse then browse to and select the Google Chrome executable.
  4. Change the Name to be Google Chrome
  5. Change the parameter count to 1
  6. Click on Configure Parameters
  7. Change Parameter Name to URL
  8. Click Apply and close the Plugin Manager

Add the plugin item:

  1. Click the Insert New Plugin Item button. This button has the lightning bold with a plus sign.
  2. Change the Display Name to samver.ca
  3. Click the Plugin drop-down and select Google Chrome
  4. Type https://samver.ca into the URL text box.
  5. Click Apply

The link to open chrome at samver.ca should now work. Simply right click the system tray icon, then click on the Chrome plugin item that should be named samver.ca
