Plugin Parameters

You have the option to give plugins parameters. By doing so, different plugin items can each have their own set of parameters when starting a program or script.

This makes it possible to start a program or script with pre-defined values, allowing for much more granular control of starting a prograrm.

Program Parameters

When a plugin is confgured as pointing to a file, the parameters are appended to as parameters to the program call.

Script Parameters

When a plugin is configured as a script, the parameters will only be inserted where indicated by the script string.

The first parameter will be inserted wherever the value {0} is used, the second parameter will be inserted wherever the value {1} is used, and so on.
