Plugin Manager

The plugin manager can be used to add new plugins to list of known plugins. Plugins can be configured to point to a specific file in the current user's file system or as a custom command-line script. Parameters can be defined so they can vary by plugin type, and menu item instance.

Example

For example, I want to add Google Chrome as a plugin, with one parameter being the website's URL I want to open. To do this I would do the following:

Create the plugin

  1. Open Plugin Manager.
  2. Click Add button.
  3. Click Browse then browse to and select the Google Chrome executable.
  4. Change the Name to be Google Chrome
  5. Change the parameter count to 1
  6. Click on Configure Parameters
  7. Change Parameter Name to URL
  8. Click Apply and close the Plugin Manager

Add the plugin item:

  1. Click the Insert New Plugin Item button. This button has the lightning bold with a plus sign.
  2. Change the Display Name to samver.ca
  3. Click the Plugin drop-down and select Google Chrome
  4. Type https://samver.ca into the URL text box.
  5. Click Apply

The link to open chrome at samver.ca should now work. Simply right click the system tray icon, then click on the Chrome plugin item that should be named samver.ca

Example - Script

Another example. I want to create a script that will copy a file from one location to another. To do this I want to make use of the batch command copy.

Create the plugin

  1. Open Plugin Manager.
  2. Click Add button.
  3. Click the Configure plugin as script check box.
  4. Enter the following into the text box: copy {0} {1}
  5. Change the Name to be Copy Script
  6. Change the parameter count to 2
  7. Click on Configure Parameters
  8. Change the Parameter 1 Parameter Name to Copy From
  9. Check the checkbox to set this parameter as required
  10. Choose Parameter 2 from the "Choose a parameter" drop down list
  11. Change the Parameter 2 Parameter Name to Copy To
  12. Check the checkbox to set this parameter as required
  13. Click Apply and close the Plugin Manager

Add the plugin item:

  1. Click the Insert New Plugin Item button. This button has the lightning bold with a plus sign.
  2. Change the Display Name to Copy File
  3. Click the Plugin drop-down and select Copy Script
  4. Add the path to a file in the Copy From parameter.
  5. Add a different path to a file in the Copy To parameter.
  6. Click Apply

When this plugin is ran, the contents of the file pointed by Copy From will be copied into the file at Copy To.
